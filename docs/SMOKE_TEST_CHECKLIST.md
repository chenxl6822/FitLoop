## 常见问题排查

| 问题 | 排查方向 |
|------|----------|
| 验证码收不到 | 检查 SMS 服务商余额、Redis 连接、后端日志 `/api/sms/send` |
| 图片上传失败 | 检查 `/uploads/` Nginx 代理、后端上传目录权限、文件大小限制 |
| 登录后闪退 | 检查 Token 解析、`local_cache.dart` 读取、后端 JWT 验证 |
| APK 安装失败 | 检查签名、Android 版本兼容、安装包完整性 |
